# game2048
在命令行界面运行的2048小游戏

本项目首发于百度文库，首发地址https://wenku.baidu.com/view/54b562e0f524ccbff12184f6

本项目原本使用的编译器为TC，如需使用gcc等其他编译器请自行替换清屏语句clrscr();为system("cls");（windows系统）

今后有可能会训练人工智能（AI）来玩本游戏，敬请期待！
